Rangkuman Materi JavaScript

16. JS Object Properties

Properti objek adalah pasangan nama dan nilai yang disimpan dalam sebuah objek. Akses properti objek bisa dilakukan menggunakan notasi titik (object.property) atau notasi kurung siku (object["property"]).

17. JS Object Methods

Metode objek adalah fungsi yang dimiliki oleh suatu objek. Metode ini bisa mengakses data internal objek menggunakan this. Contoh: 

const person = {
    name: "John",
    greet: function() {
        return "Halo, " + this.name;
    }
};

18. JS Object Display

Untuk menampilkan objek dalam JavaScript, gunakan console.log() untuk debugging, atau JSON.stringify() untuk mengubah objek menjadi string JSON.

19. JS Object Constructors

Object constructor adalah fungsi khusus yang digunakan untuk membuat banyak objek dengan struktur yang sama. Contoh: 

function Person(name, age) {
    this.name = name;
    this.age = age;
}

20. JS Events

Event adalah aksi yang terjadi di halaman, seperti klik tombol atau scroll. JavaScript dapat menangkap event ini menggunakan addEventListener. Contoh: 

button.addEventListener("click", function() {
    alert("Button clicked!");
});

21. JS Strings

String adalah tipe data teks dalam JavaScript yang diapit oleh tanda kutip ganda (" ") atau tunggal (' '). String dapat menyimpan teks, angka, atau simbol.

22. JS String Methods

JavaScript menyediakan metode untuk memanipulasi string, seperti:

  • length: Menghitung panjang string.
  • toUpperCase(): Mengubah semua huruf menjadi kapital.
  • substring(): Mengambil bagian dari string.

23. JS String Search

Metode pencarian string memungkinkan kita mencari kata atau karakter tertentu, seperti:

  • indexOf(): Mengembalikan indeks pertama dari teks tertentu.
  • search(): Mencari pola atau string.

24. JS String Templates

Template literals menggunakan backticks (`) dan dapat menyisipkan variabel atau ekspresi dengan ${}. Contoh: 

let name = "John";
console.log(Halo, ${name});

25. JS Numbers

Tipe data angka digunakan untuk menyimpan bilangan bulat, desimal, atau nilai khusus seperti Infinity dan NaN.

26. JS BigInt

Tipe data BigInt digunakan untuk angka yang lebih besar dari Number.MAX_SAFE_INTEGER. Contoh: 

let big = 123456789012345678901234567890n;

27. JS Number Methods

JavaScript memiliki metode untuk angka, seperti:

  • toString(): Mengubah angka menjadi string.
  • toFixed(): Membatasi jumlah angka desimal.
  • parseFloat(): Mengubah string menjadi angka desimal.

28. JS Number Properties

Properti angka mencakup:

  • MAX_VALUE: Angka terbesar yang dapat direpresentasikan.
  • MIN_VALUE: Angka terkecil positif.
  • NaN: Not-a-Number.

29. JS Arrays

Array adalah struktur data yang digunakan untuk menyimpan banyak nilai dalam satu variabel. Contoh: 

let fruits = ["Apple", "Banana", "Mango"];

30. JS Array Methods

Metode array meliputi:

  • push(): Menambahkan elemen ke akhir array.
  • pop(): Menghapus elemen terakhir dari array.
  • slice(): Mengambil elemen dari array tanpa mengubah array asli.
