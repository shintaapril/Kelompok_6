Rangkuman Materi JavaScript

46. JS Loop For In

Iterasi properti dalam sebuah objek:

for (let key in object) {
    console.log(key); // Mengakses nama properti
    console.log(object[key]); // Mengakses nilai properti
}

47. JS Loop For Of

Iterasi elemen dalam objek yang dapat diiterasi:

for (let element of iterable) {
    console.log(element);
}

48. JS Loop While

Menjalankan blok kode selama kondisi bernilai true:

while (condition) {
    // code
}

49. JS Break

Menghentikan loop atau switch sebelum selesai:

for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i);
}

50. JS Iterables

Struktur data seperti Array atau String yang dapat diiterasi, digunakan dengan for...of.

51. JS Sets

Kumpulan nilai unik tanpa duplikasi:

const mySet = new Set();
mySet.add(1);
mySet.add(2);
console.log(mySet.has(1)); // true

52. JS Set Methods

Metode untuk bekerja dengan Set:

  • add(value): Menambahkan elemen.
  • delete(value): Menghapus elemen.
  • has(value): Memeriksa keberadaan elemen.
  • clear(): Menghapus semua elemen.

53. JS Maps

Struktur data untuk menyimpan pasangan key-value:

const myMap = new Map();
myMap.set("key1", "value1");
console.log(myMap.get("key1")); // "value1"

54. JS Map Methods

Metode untuk bekerja dengan Map:

  • set(key, value): Menambahkan pasangan key-value.
  • get(key): Mengambil nilai berdasarkan kunci.
  • delete(key): Menghapus pasangan.
  • has(key): Memeriksa keberadaan kunci.

55. JS Typeof

Operator untuk memeriksa tipe data:

    console.log(typeof 123); // "number"
    console.log(typeof "hello"); // "string"

56. JS Type Conversion

Mengubah tipe data:

    // String ke Number
    let num = Number("123");
    console.log(num); // 123
    
    // Number ke String
    let str = String(123);
    console.log(str); // "123"

57. JS Destructuring

Ekstraksi nilai dari array atau objek ke dalam variabel:

    // Array destructuring
    const [a, b] = [1, 2];
    console.log(a, b); // 1, 2
    
    // Object destructuring
    const {name, age} = {name: "John", age: 30};
    console.log(name, age); // John, 30

58. JS Bitwise

Operator yang bekerja pada representasi biner angka:

    console.log(5 & 1); // AND -> 1
    console.log(5 | 1); // OR -> 5
    console.log(5 ^ 1); // XOR -> 4

59. JS RegExp (Regular Expressions)

Pola untuk mencocokkan string:

    const regex = /hello/i;
    console.log(regex.test("Hello")); // true

60. JS Precedence

Urutan operator dievaluasi:

    let result = 10 + 2 * 3; // 10 + (2 * 3) = 16
