Rangkuman Materi JavaScript

1. JS Introduction

JavaScript adalah bahasa pemrograman yang digunakan di web untuk membuat halaman menjadi lebih interaktif. Dapat digunakan di frontend (browser) dan backend (Node.js).

JavaScript dapat mengubah atribut HTML. Contohnya:

<button onclick="document.getElementById('myImage').src='https://via.placeholder.com/150?text=ON'">Turn on the light</button>

<img id="myImage" src="https://via.placeholder.com/150?text=OFF" style="width: 100px" />

<button onclick="document.getElementById('myImage').src='https://via.placeholder.com/150?text=OFF'">Turn off the light</button>

2. JS Where To

JavaScript dapat diletakkan di:

  • Inline: Dalam atribut elemen HTML seperti onclick.
  • Internal: Dalam tag <script> di dalam file HTML.
  • External: Di file .js yang terpisah.

3. JS Output

JavaScript output adalah cara untuk menampilkan data hasil kode kepada pengguna.

  • document.write(): Menulis langsung ke halaman.
  • alert(): Menampilkan pesan pop-up.
  • console.log(): Menampilkan output di konsol browser.
  • Manipulasi DOM seperti innerHTML.

4. JS Statements

Pernyataan adalah instruksi, biasanya diakhiri dengan titik koma (;).

let nama = "John";
console.log(nama);

5. JS Syntax

  • Gunakan titik koma (;) untuk mengakhiri pernyataan.
  • Case-sensitive (huruf besar dan kecil dibedakan).

6. JS Comments

Komentar adalah teks dalam kode yang tidak dieksekusi.

// Komentar satu baris
/* Komentar 
   banyak baris */

7. JS Variables

Variabel digunakan untuk menyimpan data:

  • var: Scope global.
  • let: Scope blok.
  • const: Nilai konstan.

8. JS Let

let umur = 25;

9. JS Const

const pi = 3.14;

10. JS Operators

  • Aritmatika: +, -, *, /.
  • Perbandingan: ==, >, <.
  • Logika: &&, ||, !.

11. JS Arithmetic

let hasil = 10 + 5; // Penjumlahan

12. JS Assignment

let x = 10;
x += 5; // x = x + 5

13. JS Data Types

  • String: "Hello".
  • Number: 42.
  • Boolean: true atau false.
  • Object: {

    14. JS Functions

    Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu dan dapat digunakan kembali.

    Contoh membuat fungsi:

    function sapa(nama) {
  return "Halo, " + nama + "!";
}

console.log(sapa("John")); // Output: Halo, John!

    Fungsi juga bisa didefinisikan sebagai ekspresi:

    const tambah = function(a, b) {
  return a + b;
};

console.log(tambah(5, 3)); // Output: 8

    15. JS Objects

    Objek adalah struktur data yang menyimpan properti (key-value pairs).

    Contoh membuat objek:

    const mobil = {
  merk: "Toyota",
  model: "Corolla",
  tahun: 2020,
  nyalakan: function() {
    console.log("Mobil menyala...");
  }
};

console.log(mobil.merk); // Output: Toyota
mobil.nyalakan(); // Output: Mobil menyala...

    Objek bisa dimanipulasi dengan menambahkan, menghapus, atau mengubah properti:

    mobil.warna = "Putih"; // Menambah properti
delete mobil.tahun; // Menghapus properti
mobil.merk = "Honda"; // Mengubah nilai
