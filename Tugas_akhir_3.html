Rangkuman Materi JavaScript

JS Array Search

Metode untuk mencari elemen dalam array:

  • indexOf(): Mengembalikan indeks elemen pertama yang ditemukan.
  • lastIndexOf(): Mengembalikan indeks elemen terakhir yang ditemukan.
  • find(): Mengembalikan elemen pertama yang memenuhi kondisi.
  • findIndex(): Mengembalikan indeks elemen pertama yang memenuhi kondisi.
let arr = [10, 20, 30];
console.log(arr.indexOf(20)); // Output: 1

JS Array Sort

Mengurutkan elemen dalam array:

  • sort(): Mengurutkan elemen secara alfabetis.
  • reverse(): Membalik urutan elemen.
let fruits = ["Banana", "Apple", "Cherry"];
fruits.sort(); // ["Apple", "Banana", "Cherry"]

JS Array Iteration

Metode untuk iterasi array:

  • forEach(): Menjalankan fungsi untuk setiap elemen.
  • map(): Mengembalikan array baru berdasarkan transformasi.
  • filter(): Mengembalikan elemen yang memenuhi kondisi.
  • reduce(): Menggabungkan nilai menjadi satu hasil.

JS Array Const

Array yang dideklarasikan dengan const tetap dapat diubah elemennya, tetapi tidak bisa dideklarasikan ulang.

const arr = [1, 2, 3];
arr[0] = 10; // Valid
arr = [4, 5, 6]; // Error

JS Dates

Gunakan Date untuk bekerja dengan tanggal dan waktu:

let today = new Date();

JS Date Formats

Format tanggal:

  • ISO: "2024-11-21T14:30:00"
  • Short: "11/21/2024"
  • Long: "November 21, 2024"

JS Date Get Methods

Metode untuk mendapatkan informasi dari objek Date:

  • getFullYear(): Tahun.
  • getMonth(): Bulan (0-11).
  • getDate(): Hari dalam bulan.

JS Date Set Methods

Metode untuk menetapkan tanggal dan waktu:

  • setFullYear(): Mengatur tahun.
  • setMonth(): Mengatur bulan.
  • setDate(): Mengatur hari dalam bulan.

JS Math

Objek Math menyediakan fungsi matematika:

  • Math.round(): Membulatkan angka ke integer terdekat.
  • Math.random(): Menghasilkan angka acak antara 0 dan 1.

JS Random

Contoh menghasilkan angka acak:

let rand = Math.floor(Math.random() * 10); // 0-9

JS Booleans

Tipe data boolean hanya memiliki nilai true atau false.

JS Comparisons

  • ==: Perbandingan nilai.
  • ===: Perbandingan nilai dan tipe data.
  • !=: Tidak sama.

JS If Else

Digunakan untuk pengambilan keputusan:

if (x > 10) {
    console.log("Besar");
} else {
    console.log("Kecil");
}

JS Switch

Digunakan untuk pengambilan keputusan dengan banyak kondisi:

switch (day) {
    case 1:
        console.log("Senin");
        break;
    case 2:
        console.log("Selasa");
        break;
}

JS Loop For

Digunakan untuk iterasi:

for (let i = 0; i < 5; i++) {
    console.log(i);
}
